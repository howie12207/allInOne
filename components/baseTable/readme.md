# 一般 Table

## 使用方法

1. 新增對應 css style

2. 新增對應 body

```html
<base-table :table-data="tableData"></base-table>
```

3. 新增對應 data

```js
tableData: {
    thead: ['期間/成果', '一年', '二年', '三年', '五年', '十年'],
    data: [
        [
            '投資價值',
            '73,427',
            '137,651',
            '199,339',
            '389,783',
            '1,240,100',
        ],
        ['投資成本', '60,000', '120,000', '180,000', '300,000', '600,000'],
        ['報酬率', '22.38%', '14.71%', '10.74%', '29.93%', '106.68%'],
    ],
},
```

若需多個表格則

1. 新增對應 css style

2. 新增對應 body

```html
<base-table :table-data="tableData"></base-table>

<base-table :table-data="tableData2"></base-table>

<!-- 以此類推 -->
```
